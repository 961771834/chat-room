# chat-room

## description

首次接触websocket,利用node+vue所做的一个聊天室demo.

## 技术栈
+ node作为服务器(express + socket.io)。
+ vue做页面

## Todos
1.  前端页面

## 运行demo.
1. cd chat-front.
2. npm i 安装前端所需依赖。
3. npm serve 前端运行开发环境。
4. cd chat-back.
5. npm i 安装后端所需依赖。
6. npm index.js.运行后端服务。
7. 在浏览器中访问http://localhost:8080;
